# Runtime storage

This directory is intentionally mostly empty in Git. Generated pages, uploads and the SQLite database are runtime artifacts and are ignored.

To run the locked Tree Planting workflow locally, copy the two PDF source standards into `backend/storage/samples/` with exactly these filenames:

- `02-Planting.pdf`
- `TREE PLANTING STANDARDS 2nd edition 2026 FINAL.pdf`

The files are not committed here because they are large binary assets.
